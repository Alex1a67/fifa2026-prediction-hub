🚀 Setup Guide

FIFA World Cup 2026 Prediction Hub - 2 steps to go live

1Enable Firestore in Firebase Console
⚠️ Without this step, NO data will save - not predictions, not champion picks, nothing.
  1. Open Firebase Console → Your Project → Firestore Database
  2. Click "Create database" if it's not created yet
  3. Choose "Start in test mode" (allows all read/write for 30 days)
  4. Pick a region - choose us-central1 (or nearest to you)
  5. Click Done
🔗 Open Firebase Console
2Set Firestore Security Rules (CRITICAL)
  1. In Firebase Console → Firestore → click the Rules tab at the top
  2. Delete everything in the editor and paste this:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
  1. Click the blue "Publish" button
  2. Wait 30 seconds for rules to propagate
✓ Once published, all predictions, champion picks, and leaderboard data will save and sync instantly.

🌐 Go Live on GitHub Pages (Free Hosting)

3Create a GitHub Repository
  1. Go to github.com/new and create a new repository
  2. Name it something like wc2026-hub - make it Public
  3. Do NOT add README or .gitignore (keep it empty)
  4. Click Create repository
4Upload Your Files to GitHub

Option A - Upload via browser (easiest):

  1. Unzip wc2026-prediction-hub-FINAL.zip
  2. In your new GitHub repo, click "uploading an existing file"
  3. Drag the entire wc2026 folder contents (all files and the css/js folders) into the upload box
  4. Scroll down → click "Commit changes"

Option B - via Git (terminal):

cd wc2026 git init git add . git commit -m "WC2026 Prediction Hub" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/wc2026-hub.git git push -u origin main
5Enable GitHub Pages
  1. In your GitHub repo, click Settings (top right tabs)
  2. Scroll down the left sidebar → click Pages
  3. Under "Source", select Deploy from a branch
  4. Branch: main · Folder: / (root)
  5. Click Save
  6. Wait 2–3 minutes, then your site is live at:
    https://YOUR_USERNAME.github.io/wc2026-hub/
✓ The URL works on any phone, laptop, or PC worldwide - just share the link!

🧪 Test Locally First (No GitHub Needed)

Open a terminal inside the wc2026 folder and run:

# Python (pre-installed on Mac/Linux): python3 -m http.server 8000 # Then open: http://localhost:8000 # OR Node.js: npx serve . # Then open the URL it shows
⚠️ Do NOT open HTML files directly (double-click). Always use a local server. Firebase CDN scripts won't load from file:// protocol.

📋 Admin Workflow - After Each Match

  1. Open /admin.html on your site
  2. Enter passphrase: admin2026
  3. Tab "Match Teams" → select the round → set which teams are in each match slot → Save
  4. Tab "Enter Results" → enter the official final score → Save Result
  5. Tab "Recalculate" → click the button → leaderboard updates instantly for all users
  6. Repeat after every match!
