Import Data ke Firebase

Upload CSV atau Excel. Data langsung masuk ke Firestore real-time.

Import CSV

Kolom: userName, matchId, teamA, teamB, predScoreA, predScoreB

Import Excel (.xlsx)

Sheet pertama dibaca. Kolom sama dengan format CSV.

Catatan Import

  • Data dengan userName + matchId sama akan diperbarui, tidak diduplikat
  • TeamA/B gunakan kode: BRA, ARG, ESP, FRA, ENG, dll.
  • MatchId format: R16-1, R32-3, QF-1, SF-1, F-1
  • Setelah import, klik Hitung Skor untuk update leaderboard
